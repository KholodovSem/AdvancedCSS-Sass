Sass 

Это препроцессор CSS, который его расширяет.
Нужен компилятор SASS.

Имеет два синтаксиса Sass & SCSS (идентичен с CSS)

Преимущества Sass:

1. Variables (переменные)
   Для этого просто перед созданием переменной используем "$".
   $primary-color: yellow;

2. Nesting - вложенность
   Делает код более читабельным, позволяет обращаться к детям,
   прямо при работе с родительским селектором.
   .nav {
    list-style: none;

    li {
      font-size: 16px;
    }
   }

   "&" in nesting 
   Записывает путь до объявленного селектора.

3. Operators (математические операторы)

4. Partials and imports (возможность делить код между файлами)

5. Mixins (Для написания повторноисользуемоего CSS-кода)
   Объявляется с помощью символа "@" и ключевого слова mixin.
   @mixin [mixinName] [($arg1, $arg2)] {
      font-size: $arg1;
      font-weight: $arg2;
   } 

   Чтобы воспользоваться созданными миксином, пишем:
   @include [mixinName][(args)];

6. Functions (Можно использовать для вычеслений)
   Объявляется с помощью символа "@" и ключевого слова function.
   @function combineValue ($a, $b) {
      return a + b + 'px'
   }

   Вызывается просто по названию.
   .nav {
      width: combineValue(50, 50);
   }

7. Extends (Позволяет наследовать общие стили от какого-то шаблона)
   Объявляется с помощью символа "%" и названия шаблона.
   %btn-placeholder {
      padding: 10px;
      text-align: center;
      border-radius: 20px;
   }

   Чтобы воспользоваться шаблоном пишем:
   .nav-btn {
      @extend btn-placeholder;
   }
   Стили можно переопределить, если это необходимо уже на самом
   элементе, который мы расширяем с помощью шаблона.

Features:

darken() - принимает цвет и процент затемнения этого цвета.
lighten() - принимает цвет и процент осветвления этого цвета.
